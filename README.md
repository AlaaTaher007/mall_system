# 🛒 Mall System (متجر إلكتروني بسيط)

A simple and efficient E-commerce web application built using **Flask** and **Supabase**. This project features a clean user interface for shoppers and a powerful dashboard for administrators.

## 🚀 Features (المميزات)
* **Product Gallery:** Browse products with images and prices.
* **Shopping Cart:** Add/remove items and calculate total price.
* **Checkout System:** Simple form to collect customer delivery info.
* **Admin Dashboard:** Secure login to manage products (Add/Edit/Delete) and track orders.
* **Supabase Integration:** Real-time database management.

## 🛠️ Tech Stack (التقنيات المستخدمة)
* **Backend:** Python (Flask)
* **Database:** Supabase (PostgreSQL)
* **Frontend:** HTML5, CSS3 (Responsive Design)
* **Environment:** Dotenv for security.
